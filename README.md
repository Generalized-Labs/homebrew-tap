# Generalized Labs Homebrew Tap

Official Homebrew tap for [Generalized Labs](https://github.com/generalized-labs) tools.

## Install

```bash
brew tap generalized-labs/tap
```

## Tools

### ironrun

Agent-safe sealed command execution. Secrets injected below AI context windows.

```bash
brew install generalized-labs/tap/ironrun
```

Verify:

```bash
ironrun version
```

More: https://github.com/generalized-labs/ironrun
